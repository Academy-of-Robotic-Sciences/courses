## About the Academy

The Academy of Robotic Sciences (ARS) provides comprehensive robotics education through a structured curriculum integrating theoretical foundations with practical applications. The program emphasizes systematic learning progression from fundamental principles to advanced topics in robotic system development.

**Version 2.0.0**: The curriculum has been transformed to academic textbook format, providing rigorous coverage of robotics theory (60-70%) balanced with practical implementation (30-40%), suitable for learners seeking in-depth technical knowledge.

### Program Characteristics

- **Theoretical Foundations** - Mathematical and physical principles underlying robotic systems
- **Systematic Progression** - Structured curriculum building from fundamentals to specialization
- **Project-Based Learning** - Applied projects reinforcing theoretical concepts
- **Competency Assessment** - Evaluation through demonstration of knowledge and skills
- **Industry-Aligned Tools** - Professional platforms and frameworks (ROS2, LeRobot, SO-100/SO-101)
- **Multiple Specializations** - Tracks in design, hardware, software, AI, and systems engineering

### Target Audience

- **Students** seeking comprehensive robotics education
- **Software engineers** expanding into robotics and embedded systems
- **Hardware engineers** adding intelligent system capabilities
- **Career transitioners** pursuing technical robotics roles
- **Self-learners** wanting structured, rigorous curriculum

## Curriculum Structure

### 1. Foundation Courses (26 hours)
Essential theoretical and practical fundamentals required for all specialization tracks.

### 2. Specialization Tracks (24-36 hours)
Choose one or more specialized study areas:
- **Robotic Designer** (60/40 theory/practice) - CAD, mechanical design, fabrication
- **Hardware Engineer** (60/40 theory/practice) - Electronics, sensors, integration
- **Software Engineer** (70/30 theory/practice) - ROS2, control algorithms, motion planning
- **AI Engineer** (70/30 theory/practice) - Computer vision, machine learning, autonomous systems
- **Systems Engineer** (70/30 theory/practice) - Architecture, integration, deployment

### 3. Capstone Projects (8-90 hours)
Comprehensive projects demonstrating integrated competencies, including optional advanced humanoid robot project.

### 4. Study Schedules
- **Full-Time** - Complete track in 1-2 weeks
- **Part-Time** - Evening and weekend sessions over 2-8 weeks
- **Self-Paced** - Flexible scheduling over 1-6 months

## Learning Pathways

### Foundational Pathway
**Recommended for:** Beginners to robotics
1. Complete foundation courses (RC-101 through RC-106)
2. Select one specialization track
3. Complete capstone project
4. Build portfolio of projects

### Comprehensive Pathway
**Recommended for:** Full-stack roboticist development
1. Foundation courses (26h)
2. Software Engineer Track (28h)
3. AI Engineer Track (36h)
4. Hardware Engineer Track (24h)
5. Optional: Humanoid Robot Project (90h)

### Career-Focused Pathway
**Recommended for:** Targeted career preparation
1. Foundation courses (26h)
2. Select tracks based on career objectives:
   - Software roles: Software + AI tracks
   - Hardware roles: Hardware + Systems tracks
   - Design roles: Designer + Hardware tracks
3. Develop professional portfolio

## Technology Platforms

### Hardware Systems
- **Robotic Manipulators**: SO-100/SO-101 robotic arms

- **Sensors**: IMU, encoders, force/torque sensors, cameras

## Getting Started

### 1. Review Prerequisites
Review foundation course descriptions to ensure adequate preparation. Most tracks require:
- Computer literacy and command-line familiarity
- Mathematical foundation (algebra, trigonometry, basic calculus)
- Time commitment for selected study schedule

### 2. Set Up Development Environment
**For complete course experience:**
- Computer running **Linux (Ubuntu 22.04+)** or WSL2
- **ROS2 Humble** (installation covered in courses)
- Access to **SO-100 robotic arm** or simulation environment
- **3D printer access** (for design track, or use simulation)

### 3. Begin with Foundations
Start with **[RC-101: Robotics Fundamentals](foundations/rc-101.md)** - the first module in the foundation curriculum.

### 4. View Interactive Content
Courses are written in **LiaScript format** for enhanced interactivity:
- Visit [LiaScript.github.io](https://liascript.github.io/)
- Upload course file or paste URL
- Access interactive quizzes, animations, and executable code

### 5. Engage with Community
- **GitHub**: Star repository, contribute via issues and pull requests
- **Portfolio**: Document projects and share learning progress
- **Open Source**: Contribute to robotics frameworks and tools
